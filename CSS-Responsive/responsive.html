Responsive

İçeriğin tarayıcınınn genişliğine göre ölçeklendirme işlemidir. Farklı tarayıcılarda sayfamızın daha iyi gözükmesi için mutlaka yapmamız gerekir. Örneğin bu çalışmadaki yazılar başlangıçta bir mobil tarayıcısına göz önüne alınmıştır. Daha sonra tarayıcının genişliğine göre font boyutu ayarlanmıştır.
